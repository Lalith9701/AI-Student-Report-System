# 🎓 AI Student Report System

An AI-powered student performance analysis system that generates reports, charts, and predictions.  
This project helps visualize student progress and assists teachers/parents in making data-driven decisions.

---

## 🚀 Features
- 📊 Student performance reports
- 📈 Data visualization with charts
- 🤖 AI-based predictions
- 📝 Easy to use interface

---

## 🛠️ Tech Stack
- **Python 3.12+**
- Flask (for web framework)
- Pandas & NumPy (for data handling)
- Matplotlib / Seaborn (for charts)
- Scikit-learn (for predictions)
